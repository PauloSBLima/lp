# 🚀 Landing Page Profissional - Paulo Lima

Este repositório contém a landing page pessoal de **Paulo Sérgio Barreiros Lima**, desenvolvida para apresentar sua trajetória profissional de forma clara e moderna.

🔗 **[Acesse o projeto aqui](https://paulosblima.github.io/lp/)**

---

## 📋 Sobre o Projeto
O objetivo desta página é servir como um portfólio digital, destacando habilidades e experiências adquiridas ao longo da minha jornada.

### ✨ Funcionalidades
- [x] Design Responsivo (Adapta-se a celulares e desktops).
- [x] Menu de navegação suave.
- [x] Seções de apresentação e contato.

---

## 🛠 Tecnologias Utilizadas
O projeto foi construído "do zero" (Vanilla), focando em performance e código limpo:

* **HTML5**: Estruturação semântica.
* **CSS3**: Estilização e layouts (Flexbox/Grid).
* **JavaScript**: Interatividade e dinamismo.
